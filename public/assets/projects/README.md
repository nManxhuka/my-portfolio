# Case-study assets

Each project has its own folder. Replace the placeholders in the case-study components with your exported screenshots, using meaningful names such as:

- `hero.webp`
- `process-research.webp`
- `feature-01.webp`
- `feature-02.webp`
- `gallery-01.webp`
- `gallery-02.webp`
- `design-system.webp`

Recommended: export WebP images at 2× the intended display size and keep individual files below 500 KB when possible.
